# Roadmap

Rough order of intent, not a commitment.

- Code-signing and notarization for the desktop builds (remove the Gatekeeper / SmartScreen warnings)
- Intel macOS build
- Multiple Copilot sessions at the same time (parallel execution across chats)
- Batch execution: run multiple tests in one shot
- Fix known issues with Record and Replay
- Exploratory Testing agent
- Integration with Atlassian Jira and Confluence (converting the Atlassian MCP to a CLI)
- Support for Azure AI Foundry (bring your own Azure deployment)
- A dedicated lightweight session for low-complexity workloads (summarization, classification)
- Workspace folder so a session can read and write files when a test needs it
- Attach to the main Chrome browser session on the machine instead of opening a separate profile
- Externalized UI config: app name, branding, CSS theme
